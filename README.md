## 📦 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/sellwawy/vite-vanilla-starter.git
cd vite-vanilla-starter
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Development Server

```bash
npm run dev
```

## 🚢 Production Build

```bash
npm run build
```
